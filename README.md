# PAYDAY 2 — Рекомендация по модам
**Version** 1.1.0

## Рекомендация

1. Удалить всех моды
2. Установить нужные моды

## Как установить DirectX Vulkan

1. Скачайте DXVK
2. Скопируйте DLL-файлы в папку игры
3. Запустите игру и проверьте
4. Если проблемы: Удалите DLL-файлы из папки игры — всё вернётся к нативному DirectX.

## Категории модов

**Античит** – моды для противодействия читерам (только для игры в открытых лобби).  
**DirectX Vulkan** – это модификация, которая заменяет DirectX 9 на Vulkan.  
**Библиотеки** – моды, обязательные для установки (зависимости других модов).  
**Графика** – моды, улучшающие графику игры (значительно снижают FPS).  
**Игровой процесс** – моды, изменяющие игровой процесс.  
**Интерфейс** – моды, затрагивающие интерфейс игры.  
**Оптимизация** – моды, исправляющие баги и оптимизирующие производительность игры.  
**Серия преступлений** – моды для игры в серию преступлений.  
**Удобство** – моды, повышающие удобство игрового процесса.
